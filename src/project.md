# Project

## [Ontology](./ontology.md)

A [scoped](./scope.md) endeavor within a [domain](./domain.md) to achieve a specific outcome. A project is a [system](./system.md) with temporal boundaries — it begins, [evolves](./evolution.md), and resolves.

A project creates [context](./context.md) for the [entities](./entity.md) within it. It bounds [focus](./focus.md) toward a defined goal, organizing resources, attention, and effort within its [scope](./scope.md).

## [Axiology](./axiology.md)

Projects channel finite [focus](./focus.md) toward outcomes. Without a project, effort dissipates across unbounded [scope](./scope.md).

A project is nested — a project may contain sub-projects, and may itself be part of a larger project. Each level has its own [scope](./scope.md) and [resolution](./resolution.md).

A project's [story](./story.md) is the sequence of decisions and changes that produce its outcome. The story holds state — what was tried, what was learned, what was decided.

## [Epistemology](./epistemology.md)

### [Cultural](./culture.md) Definition

> <a href="https://en.wiktionary.org/wiki/project" target="_blank">Wiktionary — project</a>
>
> A planned endeavor, usually with a specific goal and accomplished in several steps or stages.

> <a href="https://en.wikipedia.org/wiki/Project" target="_blank">Wikipedia — Project</a>
>
> A project is any undertaking, carried out individually or collaboratively, possibly involving research or design, that is carefully planned to achieve a particular aim.

> <a href="https://www.etymonline.com/word/project" target="_blank">Etymonline — project</a>
>
> From Latin *projectum* "something thrown forth," from *pro-* "forward" + *jacere* "to throw."

### [Pattern](./pattern.md) Expression

Project is a [pattern](./pattern.md) that is applicable anywhere in [Existence](./existence.md). A cell dividing is a project. A bird building a nest is a project. A community forming around a shared purpose is a project. A codebase evolving toward a release is a project.

All projects share: bounded scope, directed focus, temporal arc, and a story of evolution.
